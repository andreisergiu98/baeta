# Function: join()

## join(paths)

> **join**(...`paths`): `string`

Join all arguments together and normalize the resulting path.
Arguments must be strings. In v0.8, non-string arguments were silently ignored. In v0.10 and up, an exception is thrown.

### Parameters

• ...**paths**: `any`[]

string paths to join.

### Returns

`string`
